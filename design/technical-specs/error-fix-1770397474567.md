# Technical Specification: Core Application Foundation (Login & Dashboard)

## Architecture Overview
The core application foundation will be a Single-Page Application (SPA) built using React, served as static assets. User authentication will be handled via Google OAuth 2.0. The client-side application will initiate the OAuth flow, receive authentication tokens, and then communicate with a backend service for token verification and application-specific session management. The UI will be constructed using `shadcn/ui` components, styled with Tailwind CSS, ensuring a modern, consistent, and responsive user experience.
